This script slices an STL file into multiple sections and visualizes the slices using PyVista.

Features
Loads an STL file.
Computes the dimensions of the mesh.
Ensures correct orientation of the mesh.
Slices the mesh into a specified number of sections.
Visualizes the original mesh and its slices with different colors.
Requirements
numpy
pyvista
Usage
Place the STL file (Rechteckrohr_verjuengt_Volumen.stl) in the script directory.
Run the script to see the slices visualized.
Script Details
Number of Slices: 10
Slice Orientation: Along the x-axis.
Colors: Basic colors are used for visual differentiation.
